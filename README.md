# Ohjelmistotekniikka harjoitustyö

## Asennusohjeet

1. Asenna riippuvuudet
```bash
poetry install
```
2. Käynnistä sovellus
```bash
poetry install
```

## Komennot

- Käynnistä sovellus
```bash
poetry run invoke start
```

- Suorita testit
```bash
poetry run invoke test
```

- Tee testikattavuusraportti
```bash
poetry run invoke coverage-report
```
